# Python MVP Archive

This folder preserves the 1.x Python implementation as a historical reference while `dev/2.0` moves to a Node.js/TypeScript-first architecture.

Contents:

- `src/`: original Python package
- `tests/`: original pytest suite
- `examples/`: legacy ADS config example
- `run_*.py` and `_runner_bootstrap.py`: legacy convenience runners
- `docs/`: Python-era architecture and support documents

This archive is intentionally not the active application entrypoint anymore. The active 2.0 runtime lives at the repository root under `apps/`, `packages/`, and `package.json`.
